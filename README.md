# TableVerbs
Курсовая работа по ТРПО "Неправильные глаголы"

## Перед использование
* Создать папку obj/ ($mkdir obj/)
* $make
* `g++ -o tableverbs /src/read.cpp /src/main.cpp`

#Использование
* ./tableverbs
* `Запустить tableverbs.exe`
